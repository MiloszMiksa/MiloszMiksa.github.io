Komendy Windows

Zarządzanie użytkownikami

Lista kont użytkowników w systemie: net user
Tworzenie konta użytkownika: net user nazwa_użytkownika /add
Ustawienie hasła użytkownika: net user nazwa_użytkownika hasło użytkownika
Wymuszenie zmiany hasła użytkownika po zalogowaniu: net user nazwa użytkownika /logonpasswordchg:yes
Zablokowanie możliwości zmiany hasła użytkownika: net user nazwa_użytkownika /passwordchg:no
Ograniczenie czasu pracy użytkownika: net user nazwa_użytkownika /times:Pn-Wt1-12 !wyjątek Niedziela w całości!
Wyświetlenie ustawień konta użytkownika: net user nazwa_użytkownika
Ustawienie daty wygaśnięcia konta użytkownika: net user nazwa_użytkownika /expires:12/06/24
Ustawienie komentarza dla konta użytkownika: net user nazwa_użytkownika /comment:"treść_komentarza"
Dodatnie pełnej nazwy konta użytkownika: net user nazwa_użytkownika /fullname:"Pełna nazwa"
Dezaktywacja konta użytkownika: net user nazwa konta /active:no
Usunięcie konta użytkownika: net user nazwa konta /del
Net accounts -pokazuje ustawienia

Zarządzanie grupami

Lista grup użytkowników w systemie: net localgroup
Tworzenie grupy: net localgroup nazwa_grupy /add
Dodanie konta użytkownika do grupy lokalnej: net localgroup nazwa grupy nazwa konta /add
